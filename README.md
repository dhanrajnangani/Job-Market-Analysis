# Job-Market-Analysis
📊 Job Market Analysis: Data Roles in the US (2024)
I built an interactive dashboard to analyze the current US data job market, covering 742 roles across 343 companies. The goal was to uncover how salary, job titles, company size, and skill demand vary across data roles.
🔍 Key insights:
Average salary: ~$101K, with top roles reaching $254K
Highest-paying roles: Data Scientist & Data Engineer
Most in-demand skills: Python, SQL, AWS, Spark, and Tableau
Company size matters: Larger and high-revenue companies consistently offer higher compensation
Data Analysts show strong demand but lower median pay compared to engineering-focused roles
💡 This project helped me strengthen my skills in data cleaning, analysis, and visualization, while turning raw job data into actionable insights for job seekers and hiring trends.
